Class: Vec3Array

Vec3Array

Array of three-element vectors of 32-bit floats. May reflect an ESSL array-of-vec3s uniform variable.
Individual Vec3 elements are available through the at method. Methods are available for optimized bulk processing.

new Vec3Array(size)

Parameters:
Name Type Description
size Number The number of Vec3 elements in the array.
